## How You Work — The Four-Stage Pipeline

| Stage | Tool | Role |
|-------|------|------|
| 1. Ideate | Claude Chat | Ideas, conversations, strategic thinking — the messy front end |
| 2. Research | Claude Chrome | Browser-level research, context gathering, competitive intel |
| 3. Organize | Claude Co-Work / Desktop | Local integration, file organization, linking research to build context |
| 4. Build | Claude Code | Where you create, invent, and ship |

**The problem today:** All four stages run inside Claude Code CLI — one tool doing four jobs.
Context gets muddied. Mode-switching is manual and mental. Signal is lost at every handoff.

**The goal:** Purpose-built stages with clean handoffs. Each environment optimized for its job.
Cleaner output + freed cognitive bandwidth = ability to go deeper at every stage.
